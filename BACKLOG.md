# Backlog

Идеи, мысли и обнаруженные баги.

---

## Идеи и возможности

<!-- Формат: - **Название**: Описание -->

- **Экспорт и импорт задач**: Добавить возможность экспортировать все задачи в JSON формат и импортировать их обратно. Функция должна сохранять все связанные данные (проекты, контексты, теги, связи между задачами).
- **Сортировка проектов на странице проектов**: Добавить возможность сортировки проектов (по имени, дате создания, количеству задач) на странице проектов
- **Переключатель видимости проектов в левом слайдере**: Показывать проекты в списке в левом слайдере под разделом проекты, чтобы можно было скрыть или показать проекты

- **Настройка раздела по умолчанию**: Сделать настройку для открытия раздела по умолчанию
- **Глаголы для быстрого добавления задач**: Добавить поддержку глаголов (Do, Build, Check и т.д.) для быстрого добавления задач через естественный язык
- **Установка PWA**: Найти как установить PWA в браузер мобильный и ПК
- **Оповещения через мессенджеры**: Добавить возможность отправки уведомлений и напоминаний о задачах через мессенджеры (Telegram, WhatsApp и др.)



## Баги

<!-- Формат: - **Описание бага**: Шаги воспроизведения / контекст -->

- **Не работают браузерные уведомления в продакш сервере**: Браузерные уведомления не срабатывают при развёртывании на продакшн-сервере
- **Устаревшие зависимости npm**: npm warn deprecated sourcemap-codec@1.4.8, source-map@0.8.0-beta.0, glob@11.1.8 — обновить до актуальных версий (@jridgewell/sourcemap-codec, текущие релизы source-map и glob)
- **Не всегда видно кнопку очистить корзину в мобильной версии**: Кнопка очистки корзины не всегда отображается на мобильных устройствах

## Выполненные идеи

<!-- Формат: - **Название**: Описание -->

- **Автоматическое время напоминания**: При добавлении напоминания автоматически по умолчанию ставится время на 1 час больше текущего
- **Закрепить верхнюю часть приложения**: Зафиксировать верхнюю часть приложения (шапку/навигацию) при прокрутке страницы, чтобы она всегда оставалась видимой
- **Просмотр задачи по клику**: Сделать клик по задаче так, чтобы открывалось окно просмотра задачи
- **Светофор подключения к бекенду**: Визуальная индикация состояния подключения к серверу (зеленый/желтый/красный) в хедере и сайдбаре
- **Убрать дату создания задачи из списка**: Убрать из списка задач отображение того что задача создана сегодня. Вместо этого добавить дату дедлайна или показать что дедлайн не задан
- **Раздел сегодня и завтра**: Добавить раздел "Сегодня и завтра", в котором собираются все задачи с дедлайном сегодня
- **Подтверждение удаления задачи**: Добавить диалог подтверждения перед удалением задачи, чтобы предотвратить случайное удаление
- **Статус active для задач**: Добавить статус active, который означает что у задачи есть дедлайн и задача не находится в статусе Next actions, waiting for, someday, completed, trash
- **Вычёркивание заголовка выполненных задач**: Визуально вычёркивать заголовок задачи (зачёркнутый текст), когда задача отмечена как выполненная
- **Изменение пароля пользователя**: Добавить возможность изменения пароля пользователем в настройках аккаунта
- **Быстрое добавление с текущим статусом**: В каждом из разделов строка быстрого добавления задачи должна добавлять задачи с текущим статусом, если добавляется в «Сегодня и завтра» — сразу выставляется дата дедлайна
