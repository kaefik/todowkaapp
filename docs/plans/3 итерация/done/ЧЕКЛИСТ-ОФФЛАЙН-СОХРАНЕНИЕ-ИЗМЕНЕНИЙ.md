## Диагностика проблем

### Если изменения не сохраняются:

**Проверьте логи:**
- [ ] `[updateTaskMutation] Starting update`
- [ ] `[localTaskChanges] Setting local change`
- [ ] `Local change saved successfully`
- [ ] `[useTask] Task data changed`
- [ ] `Local change found`

**Используйте инструмент:**
- [ ] `window.localTaskDebug.getAll()` → показывает изменения?
- [ ] `window.localTaskDebug.get('task-id')` → показывает изменение?

**Проверьте IndexedDB:**
```javascript
indexedDB.open('todowka-local-changes').then(db => {
  const tx = db.transaction('local_task_changes', 'readonly')
  tx.objectStore('local_task_changes').getAll().onsuccess = (e) => {
    console.table(e.target.result)
  }
})
```

### Если индикатор не появляется:

- [ ] Бэкенд остановлен?
- [ ] DevTools → Network → "Offline" режим включен?
- [ ] Перезагрузили страницу?

### Если изменения исчезают после синхронизации:

- [ ] Проверьте логи: `[updateTaskMutation] Update successful`
- [ ] Проверьте логи: `clearing local changes`
- [ ] Проверьте логи бэкенда: изменения сохранены?
